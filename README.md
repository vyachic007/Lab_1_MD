# Лабораторная работа №1 - Работа с Activity в Android

## 📱 О приложении
Приложение демонстрирует базовые возможности Android SDK для работы с Activity:
- Создание и запуск Activity
- Передача данных между Activity через Intent
- Сохранение состояния при повороте экрана
- Отслеживание жизненного цикла Activity
- Явные и неявные Intent

## 🏗 Структура приложения

### 1. MainActivity (Главный экран)
**Компоненты:**
- `TextView` authorsInfo - информация об авторе (ФИО, группа)
- `Button` colorBtn - генерация случайного цвета
- `Button` sendColorBtn - отправка цвета во вторую Activity
- `EditText` switchCaseEdit - поле ввода текста
- `Button` sendTextBtn - отправка текста в третью Activity
- `Button` openLinkBtn - открытие ссылки в браузере

**Функционал:**
- Генерация случайного RGB цвета при нажатии на colorBtn
- Передача цвета в ColoredActivity через явный Intent
- Передача текста в SwitchCaseActivity
- Открытие сайта https://oreluniver.ru через неявный Intent
- Сохранение состояния (цвет и текст) при повороте экрана
- Логирование всех событий жизненного цикла

### 2. ColoredActivity (Второй экран)
**Компоненты:**
- LinearLayout для фона
- Button для возврата

**Функционал:**
- Получение цвета из MainActivity
- Установка цвета фона
- Инвертирование цвета для кнопки (негатив)
- Возврат результата "Ок" в MainActivity

### 3. SwitchCaseActivity (Третий экран)
**Компоненты:**
- Два TextView для отображения текста
- Button для возврата результата

**Функционал:**
- Получение текста из MainActivity
- Отображение исходного текста
- Отображение текста в верхнем регистре
- Возврат результата в MainActivity
